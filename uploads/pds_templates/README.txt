PDS TEMPLATE INSTRUCTIONS
========================

This folder contains the Personal Data Sheet (PDS) template for job applicants.

FILES:
- PDS_Template.html - Interactive HTML version that can be filled out and saved as PDF

HOW TO USE:
1. Open PDS_Template.html in your web browser
2. Fill out all required information
3. Print to PDF or save as PDF using your browser's print function
4. Upload the completed PDF when applying for jobs

REQUIREMENTS:
- All fields must be filled out completely and accurately
- Use black ink if printing and filling by hand
- Ensure all information is legible and correct
- Submit along with resume and cover letter

For technical support, contact the HR department.
